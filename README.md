# SchulmanagerOffline

Diese Handy-App (Android) ist eine **inoffizielle Anwendung**, um Fehler zu beheben und die Nutzung von Schulmanager Online zu verbessern.
Privates Projekt von GruenerAdler.

---

## ⚠️ Hinweis / Disclaimer

Diese App steht **in keiner Verbindung zu Schulmanager Online** und wird nicht offiziell unterstützt.

Die Nutzung erfolgt auf eigene Verantwortung:

* Du bist selbst verantwortlich für deinen Account und deine Daten
* Du musst die Nutzungsbedingungen von Schulmanager Online einhalten
* Der Entwickler übernimmt **keine Haftung** für Schäden, Datenverlust oder Account-Probleme

⚠️ Diese App verändert das Verhalten und die Darstellung einer externen Website.
Verwendung erfolgt ausschließlich auf eigenes Risiko.

---

## 📱 Nutzung

Diese App ist nur für den privaten Gebrauch gedacht.
Eine öffentliche Weiterverbreitung erfolgt auf eigene Verantwortung.

## 🛠️ Entwicklung

### Build

```
eas build --platform android --profile preview
```

### Lokal starten

```
npx expo run:android
```
